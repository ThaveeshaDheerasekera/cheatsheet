--- General Terminal Commands ---

*** ignore []

Open file with the default application:
	open [path/to/file]

Open file with a certain application:
	open -a "[Application Name]" [path/to/file]
* [-a] to specify the application

Move file
	mv [path/to/file] [new/path]

Rename file
	mv [file-name] [new-file-name]
